# flux-git-utils

Some util scripts for git

## Installation

```shell
./bin/install-to-home-local-bin.sh
```

## Commands

### git-clean-branches

```shell
git-clean-branches
```

### git-clean-ignored

```shell
git-clean-ignored /path/to/folder
```

### git-diff

```shell
git-diff /path/to/folder
```
