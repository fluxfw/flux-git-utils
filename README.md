# flux-git-utils

Some util scripts for git

## Installation

Add the files in [bin/PATH/host](bin/PATH/host) to your PATH

## Commands

### git-clean-branches

```shell
git-clean-branches /path/to/folder
```

### git-clean-ignored

```shell
git-clean-ignored /path/to/folder
```

### git-diff

```shell
git-diff /path/to/folder
```

### git-show

```shell
git-show /path/to/folder
```
